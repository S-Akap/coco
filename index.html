サービスを考えること

COCOはWebサイト制作組とUIデザイン制作組で共同で
一つのサービスを考えるグループ制作の卓です。
グループ内でもグループ外でもお互いにフィードバックし合うことで、
より良いユーザー体験を意識したサービス作りを目指します。
最終的な成果物は主にアプリのプロトタイプと
サービスのWEBサイトになります。

UX Flow

UXデザインプロセスを用いてグループでアイデアを練りました！
ここではサービスの体験価値が生まれるまで行った過程を説明します

  1. UX FLOW

    01

    アイデアの発散

    ブレインストーミングという手法を用いてみんなでキーワードに対するイメージや問題などを出し合ってアイデア出しを行いました。出てきた付箋の数はなんと30分で約400枚!!!

  2. UX FLOW

    02

    ユーザー調査

    アイデアに対して予想で制作するのではなく、根拠を持った戦略を立てるために、アンケート手法による定量調査とインタビューを行いました。
    ユーザーがどんな悩みを抱えているのか探ります！

  3. UX FLOW

    03

    調査分析/価値マップ作成

    KA法という手法でインタビューの結果を分析し、ユーザーが求めている本質的なニーズを考えどんなアイデアであれば解決になるのか考えました。

  4. UX FLOW

    04

    課題決定/ペルソナ作成

    いままでの調査でわかったことをもとにペルソナとユーザーの利用ストーリーの作成をしました。
    グループ内で共通イメージを持ち、より明確なアイデア出しに繋げました。
